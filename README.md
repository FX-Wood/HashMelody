# Hash Melody
Hash melody takes text as input and deterministically outputs a melody. It's like a melodic fingerprint!

## Requirements
- Requires python3
- tested on ubuntu 18.04

## Getting Started
```Shell
$ pip install -r requirements.txt

$ main.py
```
